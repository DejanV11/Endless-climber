Play the game on [Unity Play](https://play.unity.com/en/games/11df992d-4939-42e5-8de9-82f743305cd4/endless-climber).

### Controls
* **Move Left/Right**: *A/D* or *Left/Right Arrow Keys*
* **Jump**: *Spacebar*

### Built With
* **Game Engine**: *Unity 6.4 (6000.4.8f1)*
* **Language**: *C#*
